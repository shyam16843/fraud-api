# 🔍 Fraud Detection REST API

Production-ready fraud detection API built with **FastAPI + Docker + XGBoost**.  
Detects credit card fraud with **90% precision** and **0.92 ROC-AUC**, with SHAP explainability identifying top fraud indicators.

**Built by:** Ghanashyam T V  
**GitHub:** github.com/shyam16843  
**Live Streamlit Demo:** https://fraud-detection-shyam.streamlit.app

---

## 🚀 Quick Start

### Step 1 — Train the model
```bash
# Install dependencies
pip install -r requirements.txt

# Download creditcard.csv from Kaggle and place it here
# https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

# Train and save model
python train_model.py
```

### Step 2 — Run with Docker
```bash
# Build and start
docker-compose up --build

# API is now live at:
# http://localhost:8000
```

### Step 3 — Test the API
```bash
# Health check
curl http://localhost:8000/health

# Predict (example legitimate transaction)
```

## 📡 API Endpoints

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/` | GET | API info and available endpoints |
| `/health` | GET | Health check + model status |
| `/predict` | POST | Single transaction fraud prediction |
| `/predict/batch` | POST | Batch prediction (max 100 transactions) |
| `/docs` | GET | Interactive Swagger UI |

## 🏗️ Tech Stack

- **API Framework:** FastAPI
- **ML Model:** XGBoost
- **Class Imbalance:** SMOTEENN resampling
- **Explainability:** SHAP (V14, V12, V3 identified as top indicators)
- **Containerisation:** Docker + docker-compose
- **Dataset:** [Kaggle Credit Card Fraud Detection](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud) (284,807 transactions, 0.17% fraud rate)

## ☁️ Deploy on AWS EC2

```bash
# On your EC2 instance (Ubuntu):
sudo apt update
sudo apt install docker.io docker-compose -y
sudo systemctl start docker

# Clone your repo
git clone https://github.com/shyam16843/YOUR-REPO-NAME
cd fraud-api

# Train model and start API
python train_model.py
docker-compose up -d

# API is now live at:
# http://YOUR-EC2-PUBLIC-IP:8000
```
